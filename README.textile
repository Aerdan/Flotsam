h1. Flotsam

Flotsam is a rich-text pastebin designed primarily for archiving Let's Plays. It intentionally omits support for certain community features, such as comments and private messages, as other platforms are more suitable for communicating with users.

h2. Requirements

You must have the following to run Flotsam:

* "Mojolicious":https://mojolicious.org
* "Mojo::Pg":https://docs.mojolicious.org/Mojo/Pg
* "Parse::BBCode":https://metacpan.org/pod/Parse::BBCode
* A PostgreSQL database
